- Zum Verbinden eines USB-Sticks mit Ihrem Handy

Schalten Sie das Bluetooth auf Ihrem Handy ein.

Gehen Sie auf Ihrem Laptop/PC in den Einstellungen auf den Reiter „Bluetooth und Geräte“. Wenn Sie sich noch nicht mit Ihrem Handy verbunden haben, wählen Sie „Gerät hinzufügen“.

Drücken Sie nun auf „Bluetooth“ und wählen Sie dann Ihr Gerät aus.

Wenn Sie es ausgewählt haben, sollte es dort unter „Weitere Geräte“ erscheinen. Dort sollte „Gekoppelt“ stehen. Falls nicht, drücken Sie auf Ihr Handy und wählen Sie „Koppeln“.

Wenn das Handy gekoppelt ist, scrollen Sie etwas nach unten bis zu dem Feld „Dateien über Bluetooth senden oder empfangen“ und drücken Sie darauf.

Dann sollte dieses Fenster erscheinen. Dort können Sie entscheiden, ob Sie eine Datei senden oder empfangen möchten. Zuerst senden wir eine Datei, also drücken wir auf „Datei senden“.

Nachdem Sie darauf gedrückt haben, öffnet sich dieses Fenster. Wählen Sie erneut Ihr Handy aus und drücken Sie auf „Weiter“.

Nun erscheint dieses Feld. Dort können Sie auf „Durchsuchen“ drücken und ein beliebiges Bild auswählen, das Sie übertragen möchten.

So sollte es bei der Auswahl aussehen. Wenn Sie ein Bild gefunden haben, drücken Sie darauf und wählen Sie „Weiter“.

Sie sollten nun die Datei im oberen Feld sehen und können auf „Weiter“ drücken.

Nun erhalten Sie auf Ihrem Handy eine Benachrichtigung, dass eine Datei eingeht. Drücken Sie dort auf „Akzeptieren“.

Nach einiger Zeit erhalten Sie diese Benachrichtigung als Bestätigung der Übertragung.

So sollte es bei einem erfolgreichen Abschluss der Übertragung auf Ihrem Laptop/PC aussehen. Dort können Sie nun einfach auf „Fertigstellen“ drücken.

*Das Bild in der Galerie auf dem Handy*

Um Bilder an einen Laptop oder PC zu senden, drücken Sie in der Galerie auf das „Teilen“-Symbol und wählen Sie „Bluetooth“ aus.

Wenn Sie darauf gedrückt haben, sollte Ihr PC erscheinen. Wählen Sie diesen aus. Wenn er nicht angezeigt wird, gehen Sie in die Einstellungen unter „Bluetooth“, suchen Sie dort Ihren Laptop/PC und wählen Sie ihn aus.

Gehen Sie zurück zu Ihrem PC und drücken Sie wieder unter Bluetooth auf „Dateien über Bluetooth senden oder empfangen“. Wählen Sie diesmal „Dateien empfangen“.

Wenn Sie nun zurück zu Ihrem Handy gegangen sind und das Bild mit Ihrem Laptop geteilt haben, also in der vorherigen Ansicht den Laptop ausgewählt haben, sollten diese Dateien erscheinen. Drücken Sie hier einfach auf „Fertigstellen“.
